What important permissions we request? 

Here, we explained all the important permissions requested by our application. These permissions are necessary for our application to perform certain functions. 

 

Access network information 

This permission allows the file manager to view the user's current network status and execute corresponding advertising strategies. This permission is needed to help users master the background data usage. 

 

Collect battery statistics 

This permission is used to get the battery usage data, in order to provide power consumption ranking in Battery Saver feature. 

 

Clean cache of applications 

This permission is required to clean cache of installed applications on your device in Junk Cleaner feature. 

 

Collect component usage statistics 

This permission is required to identify the applications that need to help users to manage apps usage 

 

Find out the space used by any package 

This permission is used to find out the space used by any package so as to provide Junk Cleaner feature. 

 

View the running apps 

This permission is used to view the running apps in order to clean up memory usage. 

 

Kill background processes 

This permission is used to stop other apps and boost your phone. 

 

Broadcast after the system finishes booting 

This permission is used to help App run automatically when the phone is restarted. 

 

Keep processor from sleeping or screen from dimming 

This permission is allowed to avoid screen dimming from the process of junk scan. 

 

Read and write phone storage space 

These permissions are used to detect and delete junk files on your devices and also used to read photographs. 

 

Accessibility 

This permission is used to close apps chosen by users which can speed up users’ devices and also we need this permission to provide power saving mode for users to prolong battery life. 

 

Precise location (GPS and network-based) and approximate location (network-based) 

These permissions are required to provide more accurate, localized services based on country regions, which serves the goal to improve the user experience. 

 

Get, authenticate and manage accounts on the device 

Read and write sync settings 

These permissions are used to help users retrieve their password through your accounts on their devices. We don’t have any access to your password. 

 

Virus scan

This permission is only used to scan your device for viruses.

 

Use full screen intent 

Implement out-of-app notification reminders. 

 

Get in-app installation software 

This permission is used for functions such as APP Manager and Junk Clean.
